# 🚀 AI Interview Prep Platform

A **Production-Ready Full Stack Gen AI Web Application** that helps users prepare for job interviews using **AI-powered resume analysis, skill gap detection, and interview question generation.**

Users can upload their resumes, analyze job descriptions, identify missing skills, generate AI-based interview questions, and download **ATS-optimized resumes**.

This project simulates a **real-world SaaS product architecture** using modern full stack technologies and generative AI.

---

# ✨ Features

- 🔐 Secure Authentication using JWT
- 🚪 Logout with Token Blacklisting
- 📄 Resume Upload & Parsing
- 🤖 AI Resume Analysis using Gemini
- 📊 Skill Gap Detection
- 💼 Job Description Analysis
- 🎯 AI Interview Question Generation
- 📑 ATS-Optimized Resume Creation
- 📥 Dynamic Resume PDF Generation
- 🧠 AI Powered Interview Report System
- 📂 Recent Reports Dashboard
